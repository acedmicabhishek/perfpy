# perfpy


### **Python**
- **Use Cases**: Test automation, test framework development, log analysis, data processing
- **Key Libraries**:
  - `paramiko` - SSH for remote test execution
  - `pyserial` - Serial port communication
  - `pyyaml` - Test configuration files
  - `pandas`/`numpy` - Data analysis
  - `matplotlib` - Result visualization
  - `pytest` - Test framework
